Prototyping phase

we carefully replicated the website design to capture all potential pain points and ensure client satisfaction with the chosen layout. This crucial step was executed using Figma.

We embarked on a rapid-fire ideation session, generating quick sketches. From these sketches, we then cherry-picked the most promising elements, weaving them into a cohesive final design that we collectively embraced.

Next, we crafted a wireframe, offering a skeletal preview of the website’s architecture

We then transitioned from wireframes to a fully-fledged prototype, bringing the layout to life. This step was crucial for us to grasp the tangible interactions users would have with the website, refining our understanding of its usability and flow.

Finding the Right Plugin

Integrating Functionality

This phase was all about adding essential features like m-Pesa powered subscriptions, event management, booking systems, blogging, and user management.

As developers, we particularly enjoyed this phase because it allowed us to write custom code in PHP and JavaScript to achieve unique results.

Configuring Local Payment Gateways

We integrated m-Pesa and various internationally recognized card payment options. This involved carefully selecting the right plugins and weaving them together to enable seamless subscriptions via m-Pesa.
